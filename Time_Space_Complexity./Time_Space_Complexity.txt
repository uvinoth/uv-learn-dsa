
Time Complexity:
----------------
- Two Conditions:
    - Eliminate constants.
    - Retain the highest ordered term.
- More time less efficiency. If less time more efficiency.
- The n value increase linearly, time also increase linearly 

Example1 :

    for i := 0; i< 0; i++ {               n+1  
        for j := 0; j< 0; j++ {           n*(n+1)  
            fmt.Println("*")                n*n
        }
        fmt.Println("*")                    n
    }

    n+1 + n*(n+1)+n+n
    n+1 + n2+n+n2+n2
    3n + 2n2 // Remove constants
    n+n2 // Retain the highest ordered

    Final : n2
